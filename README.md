# SNES_A/V_Connector
This is another version for a SNES AV connector. This one is shorter and can be used as a drop replacement for the orginal socket one.

* PCB must be 1.6mm, Hardgold and beveled edges
* A M2.5x10mm screw is needed for holding the PCB in the housing.
* 3DPrinting must be mechanical resistant enought (like ABS or PETG)
